Как установить, предварительные пляски(ласки) с бубном.


Для того, чтоб приложение при запуске обнаружило наши демо-тесты, 
необходимо разпаковать tests.rar в корень нашей файловой системы:

На Android 4.2 это SD карта.
На Android 4.4 это внутрення память телефона.

В дальнейшем это будет автоматизировано при установке. Но пока что так.


Содержание архива:
- app-release.apk	- файл приложения Just Quizz
- tests.rar			- папка с тестами(демо-вариант)
- readMe.txt		- этот файлик с описанием как установить.
